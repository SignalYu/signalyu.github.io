Boyer Moore 投票算法逻辑步骤#

  1. 初始化候选人（candidate）和计票器（counter）；
  2. 遍历数组寻找可能为最大元素的候选人：
    1. count == 0 --> candidate = current element; count = 0
    2. (current element == candidate) ? count++ : count--
  3. 遍历数组验证候选人是否为多数元素（majority element）：
    1. count = 0
    2. (current element == candidate) ? count++
    3. 判断 count 是否过半
Boyer Moore 投票算法核心：只要数组中某元素出现次数过半，则遍历完毕后 candidate 变量必然指向该元素。

int majorityElement(int const nums[], int size) {
    // 初始化候选人和计票器
    int candidate = 0;
    int count = 0;

    // 遍历数组确定候选人
    for (int i = 0; i < size; i++) {
        if (count == 0) {
            candidate = nums[i];
            count++;
        } else if (nums[i] == candidate) {
            count++;
        } else {
            count--;
        }
    }

    // 遍历数组判断候选人是否为多数元素
    count = 0;
    for (int i = 0; i < size; i++) {
        if (nums[i] == candidate) {
            count++;
        }
    }
    // 存在多数元素
    if (count > size / 2) {
        return candidate;
    }
    // 不存在多数元素
    return -1;
}

#include <stdio.h>
int main() {
    int nums1[] = {3, 2, 3};
    int numsSize1 = sizeof(nums1) / sizeof(nums1[0]);
    printf("Majority element in nums1: %d\n", majorityElement(nums1, numsSize1));

    int nums2[] = {2, 2, 1, 1, 1, 2, 2};
    int numsSize2 = sizeof(nums2) / sizeof(nums2[0]);
    printf("Majority element in nums2: %d\n", majorityElement(nums2, numsSize2));
}

Majority element in nums1: 3
Majority element in nums2: 2
